# Ekstrakurikuler SMKN 1 Kawali
Ekstrakurikuler SMKN 1 Kawali merupakan web yang terdapat beberapa fitur yang menarik diantarannya absensi, laporan data absen dan masih banyak lagi.
Web Ekstrakurikuler SMKN 1 Kawali dibangun dengan menggunakan HTML, CSS , Jquery dan SweetAlert 2.
Web ini di design tannpa berpindah halaman dahulu karena menggunakan teknik AJAX
